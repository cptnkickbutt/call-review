# callreview

Internal call recording review system.

## Features in this starter

- Scan CX and VIPVoice recording sources
- Register calls in SQLite
- Detect stable files before processing
- Basic processing pipeline placeholder
- Move CX files into dated archive folders
- Keep VIPVoice files in place if already organized
- Basic web UI for listing and viewing calls
